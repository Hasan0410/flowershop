# 🌸 Flower Shop

A beautiful, responsive e-commerce website for a flower shop built with HTML, CSS, and modern web design practices.

## Project Overview

This is a static website for a flower shop featuring fresh flowers and natural products. The site includes sections for browsing products, learning about the business, customer reviews, and contact information.

## Features

- **Responsive Design** - Mobile-friendly layout with hamburger menu for smaller screens
- **Modern UI** - Clean and elegant design with a pink accent color theme
- **Navigation** - Easy-to-use navigation menu with quick links to all sections
- **Multiple Sections**:
  - Home - Hero section with call-to-action
  - About - Information about the flower shop with video
  - Products - Showcase of available flowers
  - Reviews - Customer testimonials
  - Contact - Get in touch section

- **User Icons** - Quick access to wishlist, shopping cart, and user account
- **Service Icons** - Highlights key benefits (Free Delivery, Secure Payments, etc.)
- **Image Gallery** - Product images and branding materials
- **Video Content** - Embedded video in the about section
